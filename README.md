> A vcard to JSON and back parser


## Install

```sh
$ npm install --save oniyi-vcard-parser
```


## Usage

```js
var oniyi-vcard-parser = require('oniyi-vcard-parser');

oniyi-vcard-parser('Rainbow');
```
